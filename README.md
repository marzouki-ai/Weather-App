## ⚡ Features

✨ Search weather by city name  
🌡️ Real-time temperature updates  
💧 Humidity tracking  
🌬️ Wind speed information  
☁️ Dynamic weather icons  
📱 Fully responsive UI  
⚠️ Error handling for invalid inputs  
⏳ Loading animation while fetching data
